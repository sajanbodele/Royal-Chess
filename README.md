# ♟️ Animated Chessboard UI

A responsive React chessboard with smooth piece animations and a clean realistic design.

## Features

- 🎨 Realistic board styling  
- ✨ Clean and modern UI  
- ⚡ Built with React  
- 📱  Fully responsive layout  
- ♟️  Smooth animated chess pieces 

## Tech Stack

- React
- Tailwind CSS
- JavaScript

## Getting Started

### Installation

1. Clone or fork the project from Magic Patterns
2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

## Component Usage

```jsx
import ChessBoard from './components/ChessBoard';

export default function App() {
  return <ChessBoard />;
}
```
